Raw to Stack

相机 Raw → Lightroom Classic 导出 → RoboTou Stacker 堆叠 · 一键自动化

工作流

  1. 输入 raw 文件夹路径
  2. 自动导入 raw 到 Lightroom Classic
  3. 全选 → 使用预设「robotou daochu」导出到子文件夹「矫正」
  4. 打开 RoboTou Stacker，设置输入文件夹为「矫正」
  5. 自动检测堆叠数量，创建蒙版
  6. 自动点击开始，等待堆叠完成

脚本文件

run.sh
主入口脚本，串联 LR + RoboTou
lr_workflow.applescript
Lightroom 导入 + 预设导出自动化
robotou_workflow.applescript
RoboTou Stacker 点击自动化
SKILL.md
Skill 定义与使用说明

使用方法

确保已安装 Lightroom Classic 和 RoboTou Stacker，并配置好导出预设「robotou daochu」。

bash scripts/run.sh /path/to/raw/files

需要 macOS 辅助功能权限（系统设置 → 隐私与安全性 → 辅助功能）
